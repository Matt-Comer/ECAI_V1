# ECAI_V1

Elite Contractor AI is a SwiftUI business management application designed for contractors and small business owners.

## Features

- Dashboard overview
- Customer management
- Customer detail view
- Job tracking
- Quote management
- AI Assistant screen
- Custom branding and UI
- SwiftUI Navigation

## Technologies Used

- Swift
- SwiftUI
- Xcode
- MVVM Architecture

## Screens

- Dashboard
- Customers
- Customer Details
- Jobs
- Quotes
- AI Assistant
